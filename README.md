# Alex Fitness Demo

## Описание проекта

Небольшое приложение с минимальным функционалом.

## Запуск

Детальное описание о запуске расписано в папках \*\*\*-laravel и \*\*\*-react.

Вкратце:

```shell
# frontend
cd alex-fitness-demo-frontend-framework7-react
npm install
npm run dev
```

```shell
# backend
docker compose up
php artisan migrate
php artisan serve
```
